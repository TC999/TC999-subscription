# TC999-subscription
[AppData 清理器](https://github.com/TC999/AppDataCleaner)文件夹描述规则订阅。TC999维护

# 使用方法
打开[folders_description.yaml](folders_description.yaml)文件，下载文件到软件根目录即可使用。

# 贡献
1. 复刻此仓库
2. 往`folders_description.yaml`添加内容
3. 提交拉取请求。
